# HomePlace — статичен сайт за недвижими имоти

Готов портфолио проект с HTML, CSS и JavaScript. Проектът работи без backend и е подходящ за GitHub Pages.

## Структура

- `index.html` — начална страница
- `imoti.html` — листинг с работещи филтри и пагинация
- `property-1.html` до `property-15.html` — отделна страница за всеки имот, без карта
- `uslugi.html` — услуги
- `service-*.html` — отделна страница за всяка услуга
- `za-nas.html` — за нас
- `kontakti.html` — контакти, без карта
- `styles.css` — всички стилове
- `app.js` — всички скриптове
- `assets/` — само снимки

## Безопасност и безплатно ползване

- Няма backend
- Няма API ключове
- Няма външни картни tile услуги
- Няма Leaflet/CDN зависимости
- Картата на началната страница е локална интерактивна визуализация с HTML/CSS/JS
- Формите са демо и не изпращат реални данни към сървър

## Функционалности

- Работещи филтри по град, тип, сделка, цена и квадратура
- Пагинация в секция „Имоти“
- Любими имоти чрез localStorage
- Респонсив мобилно меню отляво надясно с бутон X
- Pop-up „Добави имот“ с контакт, информация за имота и preview на снимки
- Отделни страници за всички имоти
- Отделни страници за всички услуги
